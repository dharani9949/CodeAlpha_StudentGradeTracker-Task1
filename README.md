## Overview of the Project

The Student Grade Tracker is a Java-based console application designed to manage and analyze student grades efficiently. The application allows users to enter student names and marks, calculate average marks, identify the highest and lowest scores, and generate a summary report.

---

## Objective

The objective of this project is to:

* Manage student academic records.
* Store student names and marks using arrays.
* Calculate average marks.
* Find the highest score.
* Find the lowest score.
* Display a complete student summary report.

---

## Features

* Input student names and marks.
* Store data using arrays.
* Calculate average marks.
* Find highest marks.
* Find lowest marks.
* Display summary report of all students.
* Console-based user interface.
